# Lyric Collect

> 歌词收藏工具

## 命令说明

- 安装依赖：`npm install`
- 前端自动编译：`npm run dev`
- 打包运行：`npm run build`

## 接口来源

> 本项目仅供学习交流使用，如无意侵犯了您的合法权益，请联系我们。

- 歌曲搜索：`http://www.kuwo.cn/api/www/search/searchMusicBykeyWord?key=黄昏&pn=2&rn=20`
- 歌词获取：`https://m.kuwo.cn/newh5/singles/songinfoandlrc?musicId=118987`
